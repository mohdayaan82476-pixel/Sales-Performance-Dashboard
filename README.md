# 📊 Sales Performance Dashboard

## 🧠 Project Overview

This project presents a comprehensive **Sales Performance Dashboard** built using Power BI. It analyzes key business metrics such as sales, profit, and order trends to provide actionable insights for decision-making.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top-performing categories and regions
* Track sales trends over time
* Calculate key business KPIs

---

## 📁 Dataset

The dataset used is a retail sales dataset containing:

* Order details
* Product categories
* Sales and profit values
* Regional information

---

## ⚙️ Tools & Technologies

* **Power BI** – Dashboard creation and visualization
* **Excel** – Data preprocessing
* **SQL (optional)** – Data extraction and querying

---

## 📊 Key Features

### 🔹 KPI Cards

* Total Sales
* Total Profit
* Average Order Value
* Order ID tracking

### 🔹 Visualizations

* 📈 Sales Trend over Time
* 📊 Sales by Category (Donut Chart)
* 📊 Profit by Category (Bar Chart)
* 🌍 Sales by Region

---

## 💡 Key Insights

* Technology category generates the highest profit
* West region contributes the most to total sales
* Sales show fluctuations across different time periods
* Certain categories have high sales but low profit

---

## 🎨 Dashboard Highlights

* Clean and minimal UI design
* Consistent color theme
* Interactive visuals for better insights
* Structured layout with KPI-focused design

---

## 📌 Conclusion

This dashboard helps businesses understand their sales performance and identify areas for improvement. It can support strategic decisions related to product categories and regional focus.
